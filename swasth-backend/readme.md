1.Install Required Packages

Run: npm install express cors pg bcrypt jsonwebtoken dotenv

Now I explain what each does:

Package: Why we need it
express: Create server
cors: Allow Angular to talk to backend
pg: Connect to PostgreSQL
bcrypt: Encrypt passwords
jsonwebtoken: Create login token
dotenv: Store secret safely


npm install axios->Axios is a popular, promise-based HTTP client used for making API requests in browsers and Node.js due to its simplicity, automatic JSON transformation, and superior error handling
